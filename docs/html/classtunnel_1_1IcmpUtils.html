Подробное описание

Статический класс-утилита для работы с сырыми ICMP пакетами и системными вызовами.

Содержит методы для инкапсуляции данных, проверки целостности (checksum) и взаимодействия с операционной системой.

Вычисляет стандартную контрольную сумму (checksum) для IP/ICMP заголовков (алгоритм RFC 1071).

Вычисленная 16-битная контрольная сумма, готовая для вставки в заголовок (icmphdr).

◆ xor_cipher()

void tunnel::IcmpUtils::xor_cipher ( std::vector< uint8_t > & data,
const std::string & key )
static

Применяет симметричное потоковое шифрование (XOR) к данным.

Функция изменяет переданный вектор на месте. Так как операция XOR обратима, повторный вызов этой функции с тем же ключом восстановит исходные данные.

Аргументы
dataВектор байт для шифрования/расшифровки.
keyСтрока-ключ, которая будет циклически применяться к данным.
